# Bank Transaction Fraud Detection  
Proyek Deteksi Transaksi Fraud dengan Machine Learning

## Deskripsi Proyek  
Proyek ini mengimplementasikan teknik machine learning untuk mendeteksi transaksi perbankan yang berpotensi merupakan fraud. Notebook mencakup proses lengkap mulai dari pemrosesan data, analisis awal, penerapan clustering, pembangunan model klasifikasi, hingga evaluasi performa.  
Seluruh eksperimen dijalankan di **Google Colab**.

## Fitur Utama  

### Pemrosesan & Eksplorasi Data  
- Memuat dataset ke Google Colab  
- Pembersihan data (missing values, encoding, normalisasi)  
- Analisis distribusi fitur dan identifikasi pola awal  

### Clustering  
- Penerapan algoritma clustering (misalnya K-Means)  
- Analisis segmentasi dan potensi anomali pada transaksi  

### Klasifikasi  
- Pembangunan model machine learning untuk deteksi fraud  
- Penanganan **imbalanced dataset**  
- Evaluasi menggunakan metrics seperti precision, recall, dan F1-score  

### Visualisasi  
- Visualisasi distribusi data  
- Kurva evaluasi dan confusion matrix  

## Kebutuhan Lingkungan  
Notebook menggunakan library berikut:  
- Scikit-learn  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- (Opsional) imbalanced-learn  

## Cara Menjalankan  
1. Buka notebook di Google Colab  
2. Unggah dataset atau hubungkan Google Drive  
3. Jalankan setiap sel secara berurutan  
4. Analisis hasil dan evaluasi performa model
